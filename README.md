## Getting Started

Install go ethereum(geth) in your system.

Run the geth

Add geth running port URL in const in web3 

web3 = new Web3(
      "https://121.0.0.1:8545"
    ); // Replace with the URL of your geth node
    
Run the development server:

```bash
npm run dev
# or
yarn dev
```
